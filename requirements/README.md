# Requirements

This repo uses a small set of always-needed Python packages, plus optional GPU frontends/providers.

## Files

- `requirements/core.txt`: core runtime deps (NumPy + HTTP + SSH)
- `requirements/dev.txt`: dev/test deps (pytest) + core
- `requirements/gpu.txt`: optional GPU deps (Triton/TileLang/FlagGems provider) — versions depend on your CUDA stack
- `requirements/lock/*.lock.txt`: fully pinned transitive locks generated by `pip-compile`

## Install

- Core + tests: `pip install -r requirements/dev.txt`
- Reproducible (pinned transitive): `pip install -r requirements/lock/dev.lock.txt`
- Optional GPU pipelines/providers: install CUDA-compatible Torch first, then `pip install -r requirements/gpu.txt`

For a quick sanity check, run: `pytest -q`.

For runtime dependency visibility, run: `python scripts/intentir.py env`.
